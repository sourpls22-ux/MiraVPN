# VPN Telegram Bot для Marzban

Telegram-бот для управления VPN ключами через Marzban API с поддержкой VLESS + Reality для обхода блокировок в России.

## Возможности

- ✅ Создание VPN ключей с протоколом VLESS + Reality
- ✅ Выдача конфигураций через Telegram
- ✅ Управление пользователями (создание, удаление, просмотр)
- ✅ Просмотр статистики использования
- ✅ Удобный интерфейс с кнопками

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте `.env` файл с вашими данными (уже настроен)

3. Убедитесь, что в Marzban создан inbound с именем "VLESS + Reality"

4. Запустите бота:
```bash
python bot.py
```

## Команды бота

- `/start` - Главное меню с кнопками
- `/create <username>` - Быстрое создание ключа (100 GB, 30 дней)
- `/list` - Список всех ключей
- `/config <username>` - Получить конфигурацию пользователя
- `/delete <username>` - Удалить ключ
- `/stats <username>` - Статистика пользователя
- `/cancel` - Отменить текущую операцию
- `/help` - Помощь

## Интерактивное создание ключа

1. Нажмите "➕ Создать ключ" в главном меню
2. Введите имя пользователя
3. Укажите лимит трафика в GB (или /skip для 100 GB)
4. Укажите срок действия в днях (или /skip для 30 дней)
5. Получите конфигурацию

## Требования

- Python 3.8+
- Marzban установлен и работает
- Inbound "VLESS + Reality" создан в Marzban
- Доступ к Marzban API (порт 8000)
